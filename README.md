# Cryptarithm Solver

Solver for cryptarithm puzzles implemented in Haskell.

## Description
This project solves cryptarithm equations by assigning digits to letters so that the equation holds.

Example:
SEND + MORE = MONEY

## Features
- Brute-force search over possible digit assignments
- Support for + and * operations
- Filtering invalid solutions (e.g. leading zeros)

## Tech
- Haskell

## How to run
1. Compile:
   ghc kryptarytmy.hs

2. Run:
   ./kryptarytmy
